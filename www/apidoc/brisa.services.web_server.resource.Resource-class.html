source code 
This fixes a bug that happened with some classes that inherit from Resource also inherit from basic types like dict. If the dict is empty, then this class would evaluate to False, giving unexpected results on codes like "if object:".

getChildWithDefault(self, uri, params=None)

source code 
Should be overwritten when needed. This method may apply (or not) some logic with the uri and parameters passed and match it with a child. For example, if a resource represents a folder, then it will use the uri and params to find the file or sub folder.
