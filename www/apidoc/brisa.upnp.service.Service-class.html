Class Service

source code

object --+
         |
        Service

Represents an UPnP device Service. Contains information about the service, such as service location, UPnP URL's (control, event sub, presentation, scpd XML).

Reference: http://upnp.org/specs/arch/UPnP-arch-DeviceArchitecture-v1.0-20080424.pdf.
