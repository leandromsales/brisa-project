Represents an UPnP Service Control. Through a service control a control point can invoke actions on devices and poll for values.

For using a ServiceControl to deploy service's methods (actions) just inherit from it and add your methods with "soap_" preceding the name. For example: 
>>> class HelloWorldServiceControl(ServiceControl):
...     def __init__(self):
...         ServiceControl.__init__(self)
...
...     def soap_HelloWorld(self):
...         return 'Hello World!'
Remember to set your method as the control attribute of some Service instance. This will make it automatically be exported. 
>>> service = Service()
>>> control = ServiceControl()
>>> service.control = control
