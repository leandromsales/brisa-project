Installing Python-BRisa


Install the dependencies:
  1. Python 2.5 or 2.6, python-setuptools - www.python.org
  2. Only one of the following:
    • (Recommended)* Python-circuits - http://trac.softcircuit.com.au/circuits/
    • Python-paste - http://pythonpaste.org
    • Python-cherrypy 3.1.1 - www.cherrypy.org
* We have python-circuits dependency packaged by us, available for download at the here.


Install latest python-brisa:
  1. Download latest python-brisa .tar.gz
  2. tar -zxvf python-brisa_<version>.tar.gz
    cd file
    sudo python setup.py install

Installing Python-BRisa applications

Media Server

Install the dependencies:
  • LightMediaScanner and Python-LightMediaScanner - http://lms.garage.maemo.org
  • dbus and python-dbus - http://dbus.freedesktop.org/doc/dbus-python/doc/tutorial.html
  • PyGTK, Python-gobject - http://www.pygtk.org/
Install the application
  1. Download latest brisa-media-server .tar.gz
  2. tar -zxvf brisa-media-server_<version>.tar.gz
    cd file
    sudo python setup.py install

Media Renderer

Install the dependencies:
  • gstreamer 0.10+ - http://gstreamer.freedesktop.org/download
  • gstreamer plugins - http://gstreamer.freedesktop.org/download
  • pygst - http://gstreamer.freedesktop.org/modules/gst-python.html
  • dbus and python-dbus - http://dbus.freedesktop.org/doc/dbus-python/doc/tutorial.html
  • PyGTK, Python-gobject - http://www.pygtk.org/
Install the application:
  1. Download latest brisa-media-renderer .tar.gz
  2. tar -zxvf brisa-media-renderer_<version>.tar.gz
    cd file
    sudo python setup.py install

Media Server plugins (extra)

  1. Download latest brisa-media-server-plugins .tar.gz
  2. tar -zxvf brisa-media-server-plugins_<version>.tar.gz
    cd file
    sudo python setup.py install

Troubleshooting

  • Cleaning up

    For cleaning up old framework configuration for a new fresh install, type in the terminal:

    brisa-conf -c
