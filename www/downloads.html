Installing


Framework

1. Install the following dependencies

  • Python 2.5, Python 2.5-setuptools (or 2.6)- www.python.org
  • Only one of the following:
    • (Recommended) Python-circuits - http://trac.softcircuit.com.au/circuits/
    • Python-paste - http://pythonpaste.org
    • Python-cherrypy 3.1.1 - www.cherrypy.org
We have python-circuits dependency packaged by us, available for download at https://garage.maemo.org/frs/?group_id=138.

2. Extract and install the framework

tar -zxvf python-brisa_<version>.tar.gz
cd python-brisa_<version>
sudo python setup.py install

Applications

Media Server

1. Install the dependencies
  • LightMediaScanner and Python-LightMediaScanner - http://lms.garage.maemo.org
  • dbus and python-dbus - http://dbus.freedesktop.org/doc/dbus-python/doc/tutorial.html
  • PyGTK, Python-gobject - http://www.pygtk.org/
2. Install the application
  1. Download latest brisa-media-server
  2. tar -zxvf brisa-media-server_<version>.tar.gz
cd brisa-media-server_<version>
sudo python setup.py install

Media Renderer

1. Install the dependencies
  • gstreamer 0.10+ - http://gstreamer.freedesktop.org/download
  • gstreamer plugins - http://gstreamer.freedesktop.org/download
  • pygst - http://gstreamer.freedesktop.org/modules/gst-python.html
  • dbus and python-dbus - http://dbus.freedesktop.org/doc/dbus-python/doc/tutorial.html
  • PyGTK, Python-gobject - http://www.pygtk.org/
2. Install the application:
  1. Download latest brisa-media-renderer
  2. tar -zxvf brisa-media-renderer_<version>.tar.gz
cd brisa-media-renderer_<version>
sudo python setup.py install

Media Server plugins (extra)

  1. Download latest brisa-media-server-plugins
  2. tar -zxvf brisa-media-server-plugins<version>.tar.gz
cd brisa-media-server-plugins<version>
sudo python setup.py install
