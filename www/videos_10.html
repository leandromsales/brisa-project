Videos (0.10.*)



In this section we have some demonstrative videos of the framework. We recommend you to watch them on fullscreen mode for seeing all details.

These videos apply to versions 0.10.* of python-brisa

UPnP plugin for canola2 browsing a Media Server

Canola2 is a media center like application for Nokia Tablets. We're writing a UPnP plugin for it which will be able to browse through media servers and choose a media renderer to play the media. At the moment, it already browses media servers and plays media directly on canola2, as demonstrated on the video above.


Creating a command line Control Point

On this example we teach you how to write a control point using BRisa. The final program is capable of searching and listing UPnP devices.


Creating a BinaryLight device

Shows how to write a BinaryLight device based on its UPnP specification.


Creating a Binary Light Control Point

Shows how to write a BinaryLight control point reusing the control point made on the first video. Also shows an example of a GUI control point.
